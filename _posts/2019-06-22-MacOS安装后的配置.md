# I、系统配置  

## 安装`XCode`  
直接在`App Store`中搜索安装就行了，就是安装完成之后要启动一次，同意他的协议

> $ xcode-select \-\-install

### 安装`zsh`  

#### 1.检查当前`shell`  

> $ echo $SHELL  

如果不是`zsh`执行下面操作，否则跳过  

#### 2.查看系统是否安装`zsh`  

> $ cat /etc/shells  

如果没有`/bin/zsh`就下载`zsh`如果有直接切换默认`shell`  

#### 3.下载`zsh`  

> $ brew install zsh  

#### 4.切换默认`shell`  

> $ chsh -s /bin/zsh  

### 安装`oh-my-zsh`  

> $ sh -c "$(curl -fsSL https://raw.github.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"  

or  

> $ sh -c "$(wget https://raw.github.com/ohmyzsh/ohmyzsh/master/tools/install.sh -O -)"  

[参考官网](https://ohmyz.sh)  

### 安装`zsh`帅气的主题`powerlevel10k`  

注意：安装之前请先安装并使用上述字体，否则部分内容乱码  

#### 1.安装`powerlevel10k`
> $ git clone https://github.com/romkatv/powerlevel10k.git $ZSH_CUSTOM/themes/powerlevel10k  

#### 2.配置  
vim ~/.zshrc
~~~
ZSH_THEME="powerlevel10k/powerlevel10k"
~~~

#### 3.重启终端后根据自己的喜好配置  

[参考github](https://github.com/romkatv/powerlevel10k)  

### 安装`zsh`插件  

#### 1.zsh-syntax-highlighting  
终端语法高亮

> $ git clone https://github.com/zsh-users/zsh-syntax-highlighting.git ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-syntax-highlighting  

![file](https://github.com/xuzheyang/xuzheyang.github.io/raw/master/_pic/2019-06-22/4.png)  

#### 2.zsh-autosuggestions  
猜想上一条同样开头的命令  

> $ git clone git://github.com/zsh-users/zsh-autosuggestions $ZSH_CUSTOM/plugins/zsh-autosuggestions

![file](https://github.com/xuzheyang/xuzheyang.github.io/raw/master/_pic/2019-06-22/5.png)  

#### 3.autojump  
自动跳转文件夹  

> brew install autojump  

![file](https://github.com/xuzheyang/xuzheyang.github.io/raw/master/_pic/2019-06-22/6.png)  

#### 4.修改配置文件  
vim ~/.zshrc    
~~~
plugins=(... autojump zsh-syntax-highlighting zsh-autosuggestions)  
[[ -s ~/.autojump/etc/profile.d/autojump.sh ]] && . ~/.autojump/etc/profile.d/autojump.sh
~~~

### 终端配色  

#### 1.下载`coreutils`  
> $ brew install coreutils

#### 2.生成配色文件  
> $ gdircolors --print-database > ~/.dir_colors

#### 3.添加配色  
> $ vim ~/.zshrc
添加下面配置  
~~~
PATH="$(brew --prefix coreutils)/libexec/gnubin:$PATH"
alias ls='ls --show-control-chars --color=auto'
eval `gdircolors -b $HOME/.dir_colors`
~~~

### vim配色  

> $ vim ~/.vimrc  
添加下面配置
~~~
colorscheme desert
syntax on
~~~

### 配置一些常用命令的别名  

> $ vim ~/.zshrc  

添加下面配置  
~~~
alias rm='rm -rf'
alias cp='cp -av'
alias ll='ls -lh'
alias la='ls -A'
alias scp='scp -r'
alias ip='ifconfig | grep inet | grep -v inet6'
~~~

### 配置`git`  

#### 配置`git`提交模版  

> $ wget https://gist.githubusercontent.com/keniushadu/5c7de811bfd77c4cd9c5cd981d401d0d/raw/e048f777bd690767c13544b63f936c1a3219da5e/commit-msg.txt -O ~/.git-commit-template.txt  
> $ git config --global commit.template ~/.git-commit-template.txt  

### 用TouchID授权sudo命令  

> $ sudo cp /etc/pam.d/sudo /etc/pam.d/sudo.bak  
> $ sudo vim /etc/pam.d/sudo  

~~~
auth       sufficient     pam_tid.so
~~~
